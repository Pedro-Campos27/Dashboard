# Dashboard de Temperatura (CSV)

Dashboard Streamlit que consome arquivos CSVs por dispositivo (um CSV por dia) e plota:
- Séries temporais 
- KPIs (completude, tempo dentro/fora, excursões)
- Heatmaps (média e % fora)
- Distribuição 

Dentro de /data/sample estão 2 arquivos CSVs prontos para serem anexados e ser consumidos pela aplicação.

A ideia é simular a chamada da API consumindo os dados do CSV ao invés de chamar a API diretamente como já conversado durante a aula com o professor.
